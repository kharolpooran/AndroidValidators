# AndroidValidators

Basic Android Validations for EditText like Email,Password,Number,Username,Spinner & Checkbox.

# Installations
```
allprojects {
		repositories {
			maven { url 'https://jitpack.io' }
		}
	}
  
  dependencies {
	        implementation 'com.github.kharolpooran:AndroidValidators:1.0'
	}
```
